Open-jACOB Draw2D v0.2 - A simple diagram drawing demo
by Andreas Herz (a.herz @ freegroup.de)
The purpose of this experiment is to see if a Visio-type of tool could be
developed in a web browser. It's not perfect, but it demonstrates that it can
be done. There is no Ajax writeback to the server in this demo, but it could
be easily added as a call in prototype.
