About the Open-jACOB eMail Client

The Webmail application is a lightweight IMAP client created with the Javamail API and Open-jACOB. The one page UI, like Gmail, features integrated folder list, message list and contact list. It is the must have demo application for a web application framework.

Enterprise edition feature list:
  • Multiple database connectors for mySQL, Oracle, SQL-Server Informix and HSQL.
  • Spam filter
  • Viren scanner
  • EMail forwarding
