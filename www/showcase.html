About the Open-jACOB Showcases

The Open-jACOB showcase (or application starter kit) shows you some of the amazing things that you can build by leveraging Java technology. Inside these walls you can get a first-hand glimpse of the exciting applications that have been created by Java developers and see how Open-jACOB is hanging the way that you interact with the Web. Explore the possibilities within.


Contact: info AT openjacob DOT org

jacob.blooper

The zero code Open-jACOB web application. You can create your own defect tracker and it is not neccessary to write any line of code.

All the business logic will be handled in the Visio like workflows - do it the smart way.

You can edit the workflows online in your InternetExplorer or Firefox with the Open-jACOB Draw2D component. -We think this is GREATE-

jacob.docfinder

Share and store your project documents with Open-jACOB DocFinder. Easy Navigation and search interface to retrieve your documents. You can upload new Docments to DocFinder via eMail attachments.

jacob.email

The Web mail application is a lightweight IMAP client created with the Javamail API and Open-jACOB. The one page UI, like Gmail, features integrated folder list, message list and contact list. It is the must have demo application for a web application framework.

jacob.VerSide

An application for tracking and prioritizing active and completed tasks or defects. jACOB VerSide is a Very Simple Defect tracker.

jacob.nanny

A simple server monitoring tool. The main goal of this application is to show the handling of the chart component, reporting and scheduling API of Open-jACOB.

jacob.herb

Now, more than ever, enterprise applications need to support multiple types of users with multiple types of interfaces. For example, an online store may require an fancy HTML front for Web customers, a WML front for wireless customers, an Open-jACOB web interface for administrators, and an XML-based Web service for suppliers (also provided by Open-jACOB).
