DigitalSimulator web is an experiment to transform the community philosophy of wikipedia to an enterprise web application. A digital circuit simulator developed by the community.


Using open standards, DigitalSimulator web does not depend on any third-party plugins and proprietary software. DigitalSimulator web works straight out of the box, no client configuration, no plugin installation, no platform dependencies - it just works!

...and last but not least:
  • YOU can extend it via JavaScript!
  • YOUR extension can be visible for ALL.
