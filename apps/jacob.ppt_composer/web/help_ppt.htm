Anlegen einer neuen Folie
  1. Klicken Sie auf den Button "Neu".
  2. Tragen Sie einen sprechenden Namen für Ihrer Folie ein.
  3. Laden Sie die deutsche und/oder englische Version Ihrer PowerPoint Folie hoch.
  4. Speichern Sie Ihre neue Folie.
Anmerkung: Die Vorschau (Bild auf der rechten Seite der Bildschirmmaske) Ihrer PowerPoint-Folie wird im Hintergrund berechnet und ist somit nicht sofort verfügbar.

Herrunterladen einer bestehenden Folie
  1. Gewünschte Folie in der Liste anklicken. Daten werden danach in der Maske angezeigt.
  2. Mit der linken Maustaste auf das Anzeigefeld klicken und in dem Menu [Anzeigen] auswählen
Man wird aufgefordert die Datein zu speichern oder mit dem entsprechenden Windowsprogramm zu öffnen.
Nachdem man die Datei geändert hat kann man diese dann wieder hochladen (siehe dazu weiter unten).

Aktualisieren einer bestehenden Folie
  1. Gewünschte Folie in der Liste anklicken. Daten werden danach in der Maske angezeigt.
  2. Klicken Sie auf den Button [Ändern]
  3. Laden Sie die deutsche und/oder englische Version Ihrer PowerPoint Folie hoch.
  4. Speichern Sie Ihre geänderte Folie.
Anmerkung: Die Vorschau (Bild auf der rechten Seite der Bildschirmmaske) Ihrer PowerPoint-Folie wird im Hintergrund berechnet und ist somit nicht sofort verfügbar.

Direktes bearbeiten einer bestehenden Folie
Amerkung: Damit Sie eine Folie direkt mit PowerPoint öffnen und auf dem Server speichern können, OHNE diese vorher auf Ihrem Rechner zwischenspeichern zu müssen, muss die WebSite auf der die Folien abgelegt sind, als "vertrauenswürde Site" in Ihrem Internet Explorer eingetragen sein.

Klicken Sie unter "Extras, Internetoptionen, Sicherheit" auf das Symbol "Vertrauenswürdige Sites" und anschließend auf die Schaltfläche "Sites". Hier geben Sie im oberen Feld die komplette Adresse inklusive Protokoll an, also beispielsweise "http://www.daimlerchrysler.de". Wenn die Adresse wie in diesem Beispiel nicht über https aufgerufen wird, entfernen Sie das Häkchen vor "Für Sites dieser Zone ist eine Serverüberprüfung (https:) erforderlich". Klicken Sie auf "Hinzufügen".
  1. Gewünschte Folie in der Liste anklicken. Daten werden danach in der Maske angezeigt.
  2. Klicken Sie auf den Button [Ändern]
  3. Mit der linken Maustaste auf das Anzeigefeld klicken und in dem Menu [Bearbeiten] auswählen
  4. Datei nicht speichern sonder mit entsprechenden Windowsprogramm bearbeiten (PowerPoint) und danach Speicher. Änderungen werden direkt zum Server gesendet.
  5. Nachdem Sie die Änderungen vorgenommen und PowerPoint beendet haben drücken Sie den [Speichern] Button.
Anmerkung: Die Vorschau (Bild auf der rechten Seite der Bildschirmmaske) Ihrer PowerPoint-Folie wird im Hintergrund berechnet und ist somit nicht sofort verfügbar.
