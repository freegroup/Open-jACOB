  You have successfully generate a Open-jACOB web application.


  Whats next?
  -----------
  
  1. Read the Getting Started Document ( http://www.openjacob.org/documents/jACOB__GettingStarted.pdf )
     Content:
      - Build the war-file of this application.
      - Deploy the war file into the jakarta tomcat servlet engine.
      - Accepting the license
      - Hot deploy a "jacapp" via eclipse into tomcat.

  2. Login into the '{applicationName}' application ( http://localhost:<MyTomcatPort>/{applicationName} )
  	 
  3. Configure the pop3 mail account in the '{applicationName}' application.
  
  4. Add some user to the '{applicationName}' email white list.
  
 	5. Send a eMail with a file attachment to the configured pop3 account.
 	
 
  At this point you have a running application for document sharing via web interface.
 
 
  Adminstration: http://localhost:<YourTomcatPort>/{applicationName}
  Customer view: http://localhost:<YourTomcatPort>/{applicationName}/application/{applicationName}/<YourAppVersion>/index.jsp
