Hinzufügen von virtuellen Maschinen

Parallels Desktop bietet mehrere Möglichkeiten zum Hinzufügen einer virtuellen Maschine: Sie können eine neue virtuelle Maschine erstellen oder die in Parallels Desktop 4 bzw. in Virtualisierungsprodukten von Drittanbietern verwendete virtuellen Maschinen registrieren.

Bevor Sie eine hinzugefügte virtuelle Maschine ausführen können, sollten Sie Parallels Desktop aktivieren.
