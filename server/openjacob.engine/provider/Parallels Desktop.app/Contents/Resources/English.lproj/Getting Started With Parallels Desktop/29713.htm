Erstellen einer neuen virtuellen Maschine

Wenn keine virtuellen Maschinen vorhanden sind, können Sie Ihre Arbeit in Parallels Desktop mit dem Erstellen einer neuen virtuellen Maschine im Assistenten Neue virtuelle Maschine beginnen.

So erstellen Sie eine neue virtuelle Maschine:

  1. Starten Sie Parallels Desktop und rufen Sie den Assistenten Neue virtuelle Maschine durch Wahl von Neue Windows-Installationim Willkommen-Fenster oder mit einem Klick auf den Button @ am unteren Rand der Liste der virtuellen Maschinen von Parallels auf.

  2. Legen Sie die CD/DVD mit dem Betriebssystem ein oder verbinden Sie die Installationsquelle mit Ihrem Mac. Geben Sie im Fenster Erkennung des Betriebssystems die Quelle der Installationsdateien an und klicken Sie auf Fortfahren. Das Betriebssystem wird automatisch erkannt und Sie werden aufgefordert, einen Benutzernamen und den Produktschlüssel für Windows (bei der Installation von Windows) oder ein Kennwort (bei der Installation von Linux) einzugeben.

    Falls sich in Ihrem Mac eine Boot Camp-Partition befindet und Sie in diesem Fenster die Option Boot Camp-Partition auswählen, dann wird der Assistent mit der Erstellung der Boot Camp virtuellen Maschine fortfahren. Weitere Informationen finden Sie im Kapitel Erstellung einer virtuellen Maschine für die Boot Camp-Partition im Parallels Desktop-Benutzerhandbuch.

    Wenn das Betriebssystem nicht automatisch erkannt werden soll, klicken Sie auf Erkennung überspringen. In diesem Fall müssen Sie das Betriebssystem manuell auswählen und den Typ der Installation selbst vorgeben. Zur Auswahl stehen die Installationsmodi Express, Typisch und Eigene.

  3. Klicken Sie nach Angabe von Benutzername und weiteren für die Installation erforderlichen Informationen auf Fortfahren.
  4. Bevor die virtuelle Maschine tatsächlich erstellt wird, können Sie noch einen Namen für die virtuelle Maschine eingeben und gemeinsam zu nutzende Ordner auswählen. Klicken Sie danach auf Erstellen.

    Hinweis: Um zusätzliche Einstellungen festzulegen, klicken Sie auf Erweitert.

  5. Der Assistent erstellt die virtuelle Maschine gemäß Ihrer Vorgaben und installiert das Betriebssystem sowie die Parallels Tools darin.

    Hinweis: Die Parallels Tools werden in folgenden Betriebssystemen automatisch installiert: Windows 7, Windows XP, Windows Vista, Windows Server 2003, Fedora 9, Red Hat Enterprise Linux 5, Ubuntu 9 und Ubuntu 8. In anderen Betriebssystemen müssen sie manuell installiert werden.

Die neu erstellte virtuelle Maschine ist dann über die Liste der virtuellen Maschinen zugänglich, über die Sie bequem alle virtuellen Maschinen verwalten können.

Beim Erstellen einer virtuellen Maschine mit Windows legt Parallels Desktop ein Administrator-Benutzerkonto mit leerem Kennwort an. Nach Abschluss der Installation des Gastbetriebssystems sollten Sie das Kennwort ändern.
