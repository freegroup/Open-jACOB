Wichtige Aufgaben in der virtuellen Maschine durchführen

Status der virtuellen Maschine ändern

Über das Zustandsmenü der virtuellen Maschine in der Statusleiste können Sie die virtuelle Maschine starten, ausschalten, in Standby schalten oder deren Zustand anderweitig verwalten. Benutzen Sie die Schaltfläche für den Zustand der virtuellen Maschine :

  • um die virtuelle Maschine zu starten, wenn sie zuvor angehalten wurde
  • um die virtuelle Maschine in Standby zu schalten, wenn sie ausgeführt wird
  • um die virtuelle Maschine fortzusetzen, wenn sie zuvor in Standby geschaltet oder unterbrochen wurde

Sie können auch auf den Abwärtspfeil neben der Zustandstaste der virtuellen Maschine klicken und den gewünschten Befehl aus dem Menü wählen.

Sie können auch die Liste mit den Parallels virtuellen Maschinen benutzen und die Steuerelemente des Menüs Virtuelle Maschine, um die virtuelle Maschine zu starten, anzuhalten, zu unterbrechen, in Standby zu schalten, fortzusetzen, zurückzusetzen oder auszuschalten.

Wechseln zwischen den Darstellungsmodi für virtuelle Maschinen

In einer ausgeführten virtuellen Maschine können Sie zwischen folgenden Darstellungsmodi wechseln:

  • Fenstermodus. In diesem Modus wird das Fenster der virtuellen Maschine als separates Fenster angezeigt. Dies ist der Standard-Darstellungsmodus.
  • Vollbildmodus. In diesem Modus nimmt das Fenster der virtuellen Maschine den gesamten physischen Bildschirm ein.
  • Coherence-Modus. In diesem Modus wird der Desktop der virtuellen Maschine ausgeblendet und die Programmfenster der virtuellen Maschine werden direkt in Mac OS X angezeigt parallel zu den Fenstern von Mac OS X.
  • Modalitätsmodus. In diesem Modus können Sie die virtuelle Maschine und die darin ausgeführten Programme beim Arbeiten in Mac OS X in einem transparenten, kleineren Fenster anzeigen. Standardmäßig wird dieses Fenster im Vordergrund aller geöffneten Fenster angezeigt, sodass Sie Ihre virtuelle Maschine während der Arbeit in Mac OS X überwachen können.
  • Crystal-Modus. Dieser Modus ähnelt dem Coherence-Modus mit ein paar Erweiterungen - alle Parallels Desktop-Kontrollelemente, Symbole und Menüs sind "unsichtbar" außer dem Parallels-Statussymbol in der Mac-Menüleiste.

Benutzen Sie das Menü Darstellungsmodus in der rechten unteren Ecke des Fensters der virtuellen Maschine, um die Darstellungsmodi zu wechseln.

Installieren von Windows-Programmen

Mit den folgenden Medien können Sie Ihre Favoriten-Windows-Programme in einer ausgeführten virtuellen Maschine installieren:

  • CDs und DVDs
  • Images von CDs oder DVDs, die mit Hilfe von Drittanbieter-Imaging-Produkten erstellt wurden

Hinweis: Parallels Desktop unterstützt die ISO- und DMG-Imagedateien und unterstützt unter Umständen CUE- und CCD-Imagedateien.

So verwenden Sie eine reale CD/DVD:

  1. Legen Sie den Datenträger in das CD/DVD-ROM-Laufwerk des Mac ein.
  2. Gehen Sie wie folgt vor:
    • Klicken Sie in der Statusleiste der virtuellen Maschine auf das CD/DVD-Symbol und wählen Sie im Kontextmenü Reale CD/DVD aus.
    • Wählen Sie im Menü Geräte die Option CD/DVD1>Reale CD/DVD aus.

So verwenden Sie das Image eines Datenträgers:

  1. Gehen Sie wie folgt vor:
    • Klicken Sie in der Statusleiste der virtuellen Maschine auf das CD/DVD-Symbol und wählen Sie im Kontextmenü Image einbinden aus.
    • Wählen Sie im Menü Geräte die Option CD/DVD 1>Image einbinden.
  2. Geben Sie die Imagedatei an.

Weitere Installationsschritte finden Sie in der Dokumentation des zu installierenden Programms.

Erstellen von VM-Snapshots

Sie können einen Snapshot erstellen, wenn die virtuelle Maschine ausgeführt wird, in Standby geschaltet oder angehalten wurde. So erstellen Sie einen Snapshot:

  1. Wählen Sie im Menü Virtuelle Maschine die Option Snapshot erstellen aus.
  2. Geben Sie im Fenster Snapshot-Parameter im Feld Snapshot-Name einen Namen und ggf. eine kurze Beschreibung für den Snapshot ein.

  3. Klicken Sie auf OK.

Zum Zugreifen auf einen erstellten Snapshot öffnen Sie die Snapshots der virtuellen Maschine, indem Sie im Menü die Option wählen oder im Hauptfenster der virtuellen Maschine in der Symbolleiste auf das Symbol für klicken.Snapshots verwaltenSnapshot verwaltenVirtuelle Maschine
