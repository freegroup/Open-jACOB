Überprüfen der Build-Nummer

Wenn Sie nicht sicher sind, welche Version und welches Build der Parallels Desktop-Software Sie verwenden, können Sie dies im Dialog Über Parallels Desktop überprüfen.

So finden Sie Ihre Version und die Build-Nummer heraus:

  1. Wählen Sie im Menü Parallels Desktop die Option Über Parallels Desktop.
  2. Das Fenster Über Parallels Desktop wird geöffnet. Im oberen Bereich des Dialogs werden Ihre Version und die Build-Nummer angezeigt.

Hinweis: Im Dialog Über Parallels Desktop können Sie außerdem nützliche Informationen zu Ihrem Exemplar von Parallels Desktop finden, darunter Lizenzinformationen und eine Verknüpfung mit der Website für den Parallels-Support.
