Optionen

Unter Optionen können Sie die folgenden Einstellungen für eine virtuelle Maschine konfigurieren:

  • Starten und Ausschalten . In diesem Bereich können Sie das Starten und Herunterfahren Ihrer virtuellen Maschine konfigurieren.
  • Optimierung . Diese Einstellungen beziehen sich auf die Performance Ihrer virtuellen Maschine.
  • Dienste . In diesem Bereich können Sie zwei Parallels Tools-spezifische Parameter verwalten: die Zeitsynchronisierung zwischen Mac OS X und dem Gastbetriebssystem Ihrer virtuellen Maschine, und das automatische Erfassen bzw. die automatische Freigabe des Mauszeigers.
  • Sicherheit . Die Einstellungen im Bereich Sicherheit bestimmen den Grad an Isolierung Ihrer virtuellen Maschine von Mac OS X.
  • Gemeinsame Ordner . In diesem Bereich können Sie die gemeinsam genutzten Ordner und Ihre Parameter verwalten.
  • Gemeinsame Profile . Diese Einstellungen können Sie so konfigurieren, dass Sie von Ihrem Windows-Desktop und den entsprechenden Ordnern in Ihrem Windows-Gastbetriebssystem direkt auf Ihren Mac OS X-Schreibtisch und einige Mac-Benutzerordner zugreifen können.
  • Gemeinsame Programme . Diese Einstellungen regeln die gemeinsame Nutzung von Programmen zwischen dem Windows-Gastbetriebssystem und Mac OS X.
  • Internet-Programme . In diesem Bereich können Sie Einstellungen so festlegen, dass ähnliche Objekte mit denselben Internet-Programmen geöffnet werden, unabhängig davon, zu welchem System sie gehören.
  • SmartMount . Die SmartMount-Funktion aktiviert die automatische Erkennung und das automatische Bereitstellen von Wechsellaufwerken in virtuellen Maschinen.
  • SmartGuard . In diesem Bereich können Sie die automatische Erstellung von Snapshots festlegen.
  • Rückgängig-Datenträger. In diesem Bereich können Sie den Rückgängig-Datenträger aktivieren und konfigurieren.
  • Coherence . In diesem Bereich können Sie eine Reihe an Coherence-spezifischen Optionen für Ihre virtuellen Maschine konfigurieren.
  • Vollbild . In diesem Bereich können Sie eine Reihe an Vollbild-spezifischen Optionen für Ihre virtuellen Maschine konfigurieren.
  • Modalität . In diesem Bereich können Sie eine Reihe an Modalität-spezifischen Optionen für Ihre virtuellen Maschine konfigurieren.
  • Gesten & Apple Remote . Nach Aktivierung der Optionen in diesem Bereich können Sie mit Apple Remote und Touchpad einige Windows-Programme in Ihrer virtuellen Maschine bedienen.
