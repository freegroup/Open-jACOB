Netzwerkfunktionen in einer virtuellen Maschine

Parallels Desktop ermöglicht die Nutzung von drei Netzwerktypen in virtuellen Maschinen:

  • Gemeinsames Netzwerk. Bei diesem Netzwerktyp kann die virtuelle Maschine die aktuellen Netzwerkverbindungen des physischen Computers verwenden.
  • Bridged Ethernet. Bei diesem Netzwerktyp kann die virtuelle Maschine einen der Netzwerkadapter des physischen Computers verwenden. Damit wird sie in Ihrem Netzwerk als separater Computer angezeigt.
  • Host-exklusives Netzwerk. Bei diesem Netzwerktyp kann die virtuelle Maschine nur auf den physischen Computer und andere virtuelle Maschinen zugreifen, die sich auf diesem Computer befinden.

Standardmäßig benutzt die virtuelle Maschine das Gemeinsame Netzwerk (Shared Network), da sich dieser Typ am einfachsten konfigurieren lässt. Der Netzwerkmodus Bridged Ethernet ist komplexer und die ordnungsgemäße Einrichtung erfordert unter Umständen den Einbezug des Systemadministrators.
