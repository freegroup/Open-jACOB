Erweitern einer vorhandenen Partition

Wenn Sie den nicht zugewiesenen Speicherplatz einer Partition hinzufügen möchten, die nicht die letzte Partition auf dieser virtuellen Festplatte ist, können Sie mit für diesen Zweck vorgesehenen Drittanbieteranwendungen (z. B. Acronis® Partition Expert) das Festplattenlaufwerk bequem neu ordnen, ohne die darauf gespeicherten Daten zu verlieren.
