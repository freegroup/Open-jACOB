Integrieren von Mac OS X und Ihrer virtuellen Maschine

Parallels Desktop 4 verfügt über eine Reihe von Funktionen, die eine optimale Integration zwischen Mac OS X und Ihrer virtuellen Maschine sicherstellen.

Der erste Schritt für die Integration Ihrer virtuellen Maschine mit Mac OS X besteht in der Installation von Parallels Tools auf der virtuellen Maschine. Bei den Parallels Tools handelt es sich um Dienstprogramme zur grundlegenden Integration von Mac OS X und den virtuellen Maschinen. Parallels Tools sind für Windows-, Linux- und Mac OS X-Gastbetriebssysteme verfügbar.

Für virtuelle Maschinen unter Windows bieten Parallels Tools eine Reihe von erweiterten Integrationsfunktionen, die bei der Arbeit mit Mac OS und Windows eine noch größere Kohärenz gewährleisten. Diese Funktionen umfassen u. a. SmartSelect, Coherence und Gemeinsam genutzte Programme.
