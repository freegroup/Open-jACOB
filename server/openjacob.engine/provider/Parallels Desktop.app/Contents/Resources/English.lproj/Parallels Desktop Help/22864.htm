Verwenden der Taste F8 in Windows virtuelle Maschinen in Mac OS X

Wenn Ihr primäres Betriebssystem Mac OS X Leopard ist, können Probleme auftreten, wenn Sie in einer Windows virtuellen Maschine die Taste F8 oder Tastenkombinationen mit Strg+Pfeiltaste verwenden. Dies hängt damit zusammen, dass die Taste F8 oder Tastenkombinationen mit Strg+Pfeiltaste in Mac OS X Leopard für das Dienstprogramm Spaces reserviert sind.

Mögliche Lösungen:

  • Klicken Sie in der Statusleiste der virtuellen Maschine auf das Symbol für Tastatur, und wählen Sie F8.
  • Ändern Sie die Tastenkombinationseinstellungen für das Dienstprogramm Spaces. Genaue Informationen hierzu finden Sie in der Dokumentation von Mac OS X Leopard.
